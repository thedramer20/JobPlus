# JobPlus Frontend

This is the React frontend for JobPlus.

## Stack

- React
- TypeScript
- Vite
- React Router
- TanStack Query
- Zustand

## Run locally

1. Copy `.env.example` to `.env`
2. Set `VITE_API_BASE_URL` to your Spring Boot backend
3. Install dependencies with `npm install`
4. Start the dev server with `npm run dev`

## Build

```bash
npm run build
```
